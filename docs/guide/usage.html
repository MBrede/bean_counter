Einrichtung der virtuellen Python Umgebung

Dieses Projekt wurde in Python erstellt, da bei Python die Pakete sich dynamisch ändern und Kompatibilitäten sich je nach Paketversion unterscheiden können bietet es sich an eine Python Virtual Environment anzubieten.

Nutzung der virtuellen Python Umgebung

Um die vordefinierte virtuelle Pythonumgebung zu aktivieren, kann folgender Konsolenbefehl genutzt werden:

source <Pfad des Projektordners>/env/bin/activate

Danach ist die virtuelle Umgebung aktiviert und die Ausführbarkeit der Skripte gewährleistet.

Beenden der virtuellen Python Umgebung

Wenn die Nutzung der Programme abgeschlossen sind kann die virtuelle Python Umgebung mit folgenden Konsolenbefehl wieder beendet werden:

deactivate
